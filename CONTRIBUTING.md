# CONTRIBUTING THE PROJECT

To contribute the ClownStation project (alongside supporting PCSX2), you need skills on making the emulator workable. You need the following tools:

- devkkitPro (has libctru and examples builtin)
- libctru
- Azahar emulator (or any Citra fork) or 3DS Hardware (experimental use only, use emulator to prevent crashes then)
- makerom & bannertool (get it manually, devkitPro doesn't has builtin) if you want to experiment CIA releases for 0.0.x series

That way, progress gets faster and with GUI and more. Also, don't forget to support PCSX2 to keep it alive both the base and this.
